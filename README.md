# Marathi Legal-Financial NLP Repository

## Linguistic Engineering for the Marathi Legal-Financial Landscape

An academic initiative to bridge the gap between standard Natural Language Processing and the nuanced complexities of Marathi administrative, legal, and financial documentation.

## Repository Structure

```
marathi-legal-financial-nlp-repository/
│
├── index.html                        # Home page
├── css/styles.css                    # Shared styles
├── assets/
│   ├── images/                       # Image assets
│   └── diagrams/                     # Workflow diagrams
│
├── pages/
│   ├── preprocessing.html            # Text Preprocessing concepts
│   ├── feature-engineering.html      # Feature Engineering concepts
│   ├── language-representation.html  # Language Representation (Word Embeddings)
│   ├── language-models.html          # Language Models (Transformers, N-grams)
│   ├── comparisons.html              # Comparative Analysis of NLP methods
│   ├── workflows.html                # NLP Workflow Architectures
│   ├── applications.html             # Real-World Applications
│   ├── research.html                 # Research & Industry Insights
│   └── sustainability.html           # Sustainability & Long-term Impact
│
└── README.md
```

## Pages Overview

- **Home** (`index.html`): Portal landing page with hero, statistics, research domains, and pipeline preview
- **Preprocessing** (`pages/preprocessing.html`): Tokenization, normalization, stopword removal for Marathi
- **Feature Engineering** (`pages/feature-engineering.html`): BoW, TF-IDF, N-gram features
- **Language Representation** (`pages/language-representation.html`): Word2Vec, FastText, IndicBERT, MuRIL
- **Language Models** (`pages/language-models.html`): Statistical, Neural, Transformer architectures
- **Comparative Analysis** (`pages/comparisons.html`): Side-by-side methodology comparisons with tables
- **Workflows** (`pages/workflows.html`): Interactive pipeline diagrams for text processing, feature extraction, and vectorization
- **Applications** (`pages/applications.html`): Legal classification, tax processing, search engine, contract analysis, banking chatbots
- **Research** (`pages/research.html`): Research breakthroughs, evolution timeline, industry implementations, open source ecosystem
- **Sustainability** (`pages/sustainability.html`): Corpus preservation, model efficiency, community governance

## Technology

- Built with Tailwind CSS via CDN
- Material Symbols for icons
- Google Fonts (Inter, Poppins, IBM Plex Sans)
- Responsive design with sidebar navigation

## Getting Started

Open `index.html` in any modern web browser to explore the research portal.
